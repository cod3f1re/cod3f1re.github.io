Game Fruit cod3f1re

La pandemia por covid 19 fue un suceso que ha dejado marca sin duda alguna en toda la historia de la humanidad, y aunque esto en un futuro será algo muy común de saber, las medidas de precaución ante el covid 19 son datos muy importantes que todos los niños pequeños deberían de saberlo, y que mejor que facilitarles el aprendizaje mediante un videojuego en línea que les permita aprender y jugar al mismo tiempo. Covidrama es un juego muy sencillo, que hace uso de html, css y javascript para simular el juego del memorama pero con dibujos sobre las medidas de prevención ante el covid 19 como el uso del cubrebocas, el gel antibacterial, etc.

Características de la aplicación web

  1. Login con firebase.
  2. Guarda y visualiza las puntuaciones del usuario en una tabla con sus respectivas fechas.
  3. Videojuego de memorama funcional, en donde al fallar .

Se desarrolló con el IDE Visual Studio Code, firebase como bd y Javascript con Css para las animaciones del videojuego.
