Prevención Covid Uni cod3f1re

Esta es una aplicación android que se pretende ser utilizada dentro de un entorno estudiantil para poder llevar un mejor control de los alumnos que presenten síntomas de covid 19 dentro de un plantel y proporcionar información importante acerca de ellos.

Características de la aplicación

  1. Registrar síntomas de covid 19 o no de un alumno.
  2. Interfaz con los principales métodos de prevención del covid 19.
  3. Interfaz con los síntomas más comunes del covid 19.
  4. Posibilidad de llamar a emergencias con solo pulsar un botón.
  5. Interfaz con un mapa de los hospitales certificados para atender a un posible paciente de covid 19.

Se desarrolló con el IDE de android studio y el lenguaje de programación java.
