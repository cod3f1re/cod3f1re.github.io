Animalia cod3f1re

Animalia es una aplicación web realizada con Php y Bootstrap, pretende ser un sistema de administración sencillo con las siguientes características.

Características de la aplicación web

  1. Interfaz de inicio para promover los servicios de la veterinaria.
  2. Gestión del módulo de Vacuna, que ofrece la posibilidad de agregar, visualizar, editar y eliminar sus registros.
  3. Gestión del módulo de Veterinarios, que ofrece la posibilidad de agregar, visualizar, editar y eliminar sus registros.
  4. Gestión del módulo de Mascotas, que ofrece la posibilidad de agregar, visualizar, editar y eliminar sus registros.

Se desarrolló con el IDE webstorm y Php en la parte del backend con MySQL como Base de datos.
